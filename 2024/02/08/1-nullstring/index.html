说透sql.NullString

本文建议阅读时间2分钟

关键词

golang | sql.NullString | NULL

名词定义

sql.NullString is a data type in the Go programming language’s database/sql package. It represents a string that may be null. This type is used when you want to handle nullable string columns in a SQL database.

sql.NullString是go语言database/sql包中的一个数据结构，代表一个可能为空的string类型值。使用场景为：当期望在数据库中，将一个字段的值设置为空

相关源码

sql.NullString实现了scan 和 value接口

常见使用场景

  • 当数据库中，字段有唯一索引，string会默认将字段值设置为空字符串，唯一索引冲突
  • 当字段是否NULL影响业务逻辑时，比如会用到 where ** is null，此时如果string，值为""，结果可能非预期

扩展延伸

除sql.NullString外，还有sql.NullTime等数据类型，区别在于time.Time的零值为”0000-00-00 00:00:00”，而mysql中，datetime的值区间为”1000-01-01 00:00:00.000000” 到 “9999-12-31 23:59:59.999999”，零值非法。
